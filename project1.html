The Vault Website

Project Overview

This is a blog style website that I created with a group using HTML, CSS, and a bit of JavaScript. The website is called "The Vault" and it is a blog that focuses on different kinds of interests. The website has a clean and modern design, with a navigation bar at the top, a main content area, and a footer at the bottom. The main content area features blog posts with images, titles, and excerpts. The footer includes links to different sections of the site and contact information.

The website is fully responsive, meaning it looks great on all devices, including desktops, tablets, and smartphones. We used media queries in the CSS to ensure that the layout adjusts appropriately for different screen sizes.

Overall, I am very proud of this project and I believe it showcases my skills in HTML and CSS. I am excited to continue learning and improving my web development skills in future projects.
