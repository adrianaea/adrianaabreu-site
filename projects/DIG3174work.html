Front-End Web Design Course Work Showcase

From DIG3174: Front-End Web Design

Project Overview

This is a showcase website that I created to display the assignments I completed in my Front-End Web Design course. The website is built using HTML, CSS, and JavaScript, featuring a clean design. The assignments are organized into a visually interactive layout that reflects both creativity and structure.

The design is inspired by a retro desktop interface, with layered window-style elements that mimic the look and feel of a classic operating system. Each module is clearly labeled and accessible by double clicking on the specific icon, allowing users to easily explore assignments as if they are on their own computer desktop.

Overall, this project showcases my growth in front-end web design and my ability to create interactive and user-friendly web interfaces. I am proud of the work I have done and am excited to continue developing my skills in this field.
