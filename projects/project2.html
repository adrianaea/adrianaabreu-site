Ariana Grande Webpage

From DIG2500: Fundamentals of Interactive Design

Project Overview

This project is an informational page about the artist Ariana Grande. It includes a brief biography, a gallery of images, and a video section featuring some of her popular songs. The page is designed with a clean and modern aesthetic, using a combination of HTML, CSS, and JavaScript to create an engaging user experience.

The color scheme is inspired by Ariana Grande's signature style, with a mix of soft pastels and bold accents. Overall, this project showcases my skills in web development and my ability to create visually appealing and user-friendly websites.
